## 发送短信验证码
```
Post /auth/v1/yw-mobile-captcha?mobile=
```

> 请求参数: mobile:手机号 



## 家长手机号登录
```
GET /nauth/record/mobileLogin
```

> 请求参数: mobile:手机号 ,captcha:验证码
> 需根据返回学生数量判断.当查出多个学生时 让家长根据学校年级班级信息 选择学校
 根据选择的学生ID 调用扫码登录接口,参考问卷链接填写时验证学生信息
